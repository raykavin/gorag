# Go Devcontainer Template

Personal **Dev Container** template for **Golang** development, focused on productivity, consistency, and a ready-to-use environment.

## Purpose

Provide a consistent Go development environment using **Dev Containers**, avoiding manual setup and machine-specific differences.

## What’s Included

- Preconfigured Go environment  
- VS Code Dev Containers support  
- Basic tools for Go development  
- Simple and reusable setup  

## How to Use

1. Clone this repository:
   ```bash
   git clone https://github.com/raykavin/devcontainer-go.git
   ```
2. Open the project in **VS Code**
3. When prompted, select **“Reopen in Container”**

Alternatively, copy the `.devcontainer` folder into any Go project and reuse the setup.

## Notes

- This is a **personal template**, meant to evolve over time  
- Go versions, tools, extensions, and project structure can be fully customized
